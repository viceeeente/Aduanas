¿Como completo el formulario de Ingreso o Salida?

Primero para poder ingresar a realizar el formulario, debes pasar por un login, el cual te pedira tu nombre de usuario y clave unica, esto se hace para tener una mayor seguiridad. Una vez el usuario haga el login, se muestra un formulario con sus datos ya establecidos, para que el formulario se guarde debe seleccionar la fecha de Ingreso al Pais (Ya sea extranjero o residente) y la fecha de Salida, tambien se muestran dos selecciones -que se pueden dejar en blanco- la de si viaja en vehiculo propio o si lleva un menor de edad de acompañante, estas opciones solo son necesarias marcarlas si el usuario las cumple, de no ser así, solo de a Enviar.

¿Que documentos debo presentar si viajo en mi vehiculo personal?

Los documentos que debe ingresar el usuario al formulario son:

  • Patente
  • Marca
  • Modelo
  • Color
  • Pais de Origen

    • Ya con esos documentos puede completar el formulario si lo requiere
